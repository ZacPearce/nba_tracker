# NBA Game Tracker 🎯

This Django web app lets users track NBA teams, players, and game logs — and save their favorite players to a personal watchlist!

## 🚀 Project Overview

NBA Tracker lets users:
- View and manage NBA teams and players
- Add/remove items from their watchlist
- Log game performances
- Register and log in securely
- View a responsive, styled UI with Bootstrap
- Deploy to the web using Heroku
This project demonstrates full-stack development with a focus on data modeling, authentication, and user interaction.


## ✅ Features

- Create/view/delete:
  - Teams
  - Players
  - Game logs
- Track favorite players in your own watchlist
- User login/logout
- Bootstrap layout + JavaScript interactions
- Deployed to Heroku with PostgreSQL

## 🛠️ Tools & Technologies

- **Python 3**
- **Django 5**
- **HTML + CSS** (with **Bootstrap 5**)
- **JavaScript** (for client-side validation & interactivity)
- **PostgreSQL** (on Heroku)
- **Gunicorn + Whitenoise** (for Heroku deployment)
- **Heroku** (for hosting)
- **Git + GitHub** (for version control)

## 🔐 Authentication Features
Custom homepage with login/register prompts
Secure Django authentication system
Logout via POST form (for safety)
Redirects after login/logout
JavaScript validation on registration form

## ✨ Extra Features
Interactive homepage with button-based explanations
Form validation (client-side) using JavaScript
Mobile-friendly design with Bootstrap grid system
