# NBA Game Tracker 🎯

This Django web app lets users track NBA teams, players, and game logs — and save their favorite players to a personal watchlist!

## ✅ Features

- Create/view/delete:
  - Teams
  - Players
  - Game logs
- Track favorite players in your own watchlist
- User login/logout
- Bootstrap layout + JavaScript interactions
- Deployed to Heroku with PostgreSQL

## 🔧 Tech Used

- Python 3 + Django
- PostgreSQL (via Heroku)
- HTML/CSS/Bootstrap
- JavaScript for interactivity
- Git + GitHub for version control
